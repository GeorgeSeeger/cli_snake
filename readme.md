__Command Line Snake__

This was made to amuse myself on a train from Kyoto to Tokyo

*CLI Snake v0.1*
Ruby 2.4.1
Linux

Simply download the repo, cd into the folder and play 

```$ ruby play.rb```

The default controls are WASD plus enter
